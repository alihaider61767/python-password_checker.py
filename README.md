# 🔐 Password Strength Checker (Python CLI)

A simple Python-based command-line tool that checks the strength of a password using multiple security criteria.

## 🚀 Features
- Checks password length
- Detects uppercase & lowercase letters
- Detects numbers
- Detects special characters
- Displays password strength level

## 🛠️ Technologies Used
- Python
- Regular Expressions (re module)
- VS Code

## ▶️ How to Run
1. Open the project folder in VS Code
2. Open terminal
3. Run the command:
   python password_checker.py

## 🎯 Purpose
This project is created to understand:
- String handling
- Functions
- Conditional logic
- Regular expressions

## 🔒 Password Criteria
- Minimum 8 characters
- Upper & lower case letters
- At least one number
- At least one special character
